Opis zbioru danych¶

Analizowany przeze mnie zbir danych 2022 Fuel Consumption Ratings zawiera dane o samochodach oraz wskaniki zuycia paliwa i szacunkow emisj dwutlenku wgla dla nowych modeli przeznaczonych do sprzeday detalicznej w Kanadzie w 2022 r. Twrc zbioru jest organizacja Natural Resources Canada (NRCan), a pochodz one ze standardowych, kontrolowanych bada laboratoryjnych i procedur analitycznych przeprowadzanych przez producentw pojazdw. Zbir zawiera 946 rekordw.

Dane pochodz z platformy Kaggle i s dostpne pod adresem: https://www.kaggle.com/datasets/rinichristy/2022-fuel-consumption-ratings

W swojej pracy dokonam analizy zuycia paliwa w miecie, na autostradzie i w cyklu mieszanym, a nastpnie sprawdz czy wystpuje korelacja midzy pojemnoci silnika a spalaniem paliwa.

central_moment <- function(dane, nth) {
  mean((dane - mean(dane)) ^ nth)
}

var_population <- function(data) {
  central_moment(data, 2)
}

sd_population <- function (data) {
  sqrt(var_population(data))
}

kurtosis <- function(data) {
  central_moment(data, 4) / sd_population(data) ^ 4
}

skewness <- function(data) {
  central_moment(data, 3) / sd_population(data) ^ 3
}

showStatistics <- function(data, title, xtitle) {

print(c(Srednia = mean(data),
        Mediana = median(data),
        Wariancja = var_population(data),
        Odchylenie = sd_population(data),
        Minimum = min(data),
        Maksimum = max(data),
        Kurtoza = kurtosis(data),
        Skosnosc = skewness(data),
        Kwantyl = quantile(data, c(0.25, 0.75))

  ))

  k <- 1 + 3.322 * log(length(data))
  #k <- sqrt(length(data))
  #k <- (max(data) - min(data)) / (2.64 * n^(1/3) * IQR(data))
  #k <- 2 * IQR(data) / length(data)^(1/3)
  hist(data, breaks = k, main = title, ylab = "Czsto", xlab = xtitle)


  intervals <- cut(data, k)
  table(intervals)
}

Testowanie hipotez¶

Na podstawie dostpnych danych postanowiem zbada czy spalanie paliwa na autostradzie i w miecie jest zrnicowane. Powszechnie mwi si, e zuycie paliwa w miecie jest wiksze. Sformuowane przez mnie hipotezy brzmi nastpujco:

Hipoteza zerowa: $\mu_m = \mu_a \Rightarrow \mu_m - \mu_a = 0$

Hipoteza alternatywna: $\mu_m > \mu_a \Rightarrow \mu_m - \mu_a > 0$

Moje populacje s sparowane, poniewa dane w parach dotycz tego samego modelu samochodu. Jeli interesuje mnie czy rednie pomidzy populacjami s rwne to tak naprawd interesuje mnie czy ich rnica jest rwna 0. Mog wic odj jeden wynik w parze od drugiego i traktowa je jako now populacj, populacj rnic wynikw.

Nowa populacja ma rozkad w przyblieniu normalny, liczebno prby jest wystarczajca, wariancja jest znana, wic mog wykona prawostronny test Z.

z_test <-function(alpha, population, sample) {

  mean_population <- mean(population)
  var_population <- var_population(population)
  mean_sample <- mean(sample)

  z_crit <- qnorm(1-alpha)

  Z <- (mean_sample - 0) / sqrt( var_population / length(population) )
  p <- 1 - pnorm(Z)

  if (Z < z_crit) {
    sprintf("Wartosc testu Z: %f. Zbior krytyczny: <%f; inf). Nie ma dowodow na odrzucenie hipotezy zerowej. P-value: %f", Z, z_crit, p)
  } else {
    sprintf("Warto testu Z: %f. Zbior krytyczny: <%f; inf). Naley odrzuci hipotez zerow. P-value: %f", Z, z_crit, p)
  }
}

Warto testu Z znalaza si w zbiorze krytycznym, wic naleao odrzuci hipotez zerow. P-value jest mniejsze ni 0.01, wic istnieje bardzo silna przesanka przeciwko hipotezie zerowej. Poziom istotnoci statystycznej zosta obrany bardzo niewielki, wic prawdopodobiestwo odrzucenia "prawdziwej hipotezy zerowej" i popenienia bdu pierwszego rodzaju byo bardzo mae. Hipoteza bya rzeczywicie prawdziwa, poniewa spalanie w miecie jest rednio wiksze ni spalanie na autostradzie, wic nie zosta popeniony bd pierwszego rodzaju.

Podsumowanie¶

  • Motywacj wyboru zbioru danych i testw bya ch zbadania, czy faktycznie istniej powszechnie znane zalenoci i uda si je wykaza wykonujc analiz statystyczn.
  • Wynikiem pierwszego testu jest bardzo dua wartoc statystyki testowej i bardzo mae p-value. Wnioskuj, e sprawdzana zaleno bya "zbyt oczywista" i nie byo szans na jej obalenie.
  • Wspczynnik korelacji pomidzy pojemnoci silnika a zuyciem paliwa okaza si wysoki i istotny statystycznie.
  • Dalszym krokiem mogoby by zbadanie zalenoci pomidzy innymi danymi w zbiorze, np. typem paliwa, mark samochodu, itp. Mona by te prbowa porwna dane ze zbiorem z innego roku.
